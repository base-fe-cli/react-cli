## 使用

```bash
$ npm install
$ npm start         # 访问 http://localhost:3000
```

## 功能

    - router
        - menuConfig.json 菜单配置文件
        - ComponentConfig.js 菜单组件映射文件
        - index.js 菜单导航生成
